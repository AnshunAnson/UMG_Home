# 聚焦视觉效果计划

## 需求理解
悬停时需要淡化文本，重点聚焦视觉效果

## 设计方案

### 1. 悬停状态设计

#### 默认状态
- 项目卡片全宽，高度 200px
- 显示完整信息：图标、标题、描述、技术标签
- 文本清晰可见

#### 悬停状态
- 当前卡片扩展到 500px 高度
- **文本淡化**：
  - 标题：透明度 30%
  - 描述：透明度 20%
  - 技术标签：透明度 25%
  - 只保留最基本的项目识别信息
- **视觉效果突出**：
  - 项目专属动画占据卡片主要区域（70%空间）
  - 动画全屏播放，不受卡片边界限制
  - 使用大图标、动态图形、粒子效果等

### 2. 视觉效果设计

#### 汽车渲染平台
**视觉效果（悬停时占主导）：**
- 大型3D汽车线框模型旋转（占卡片60%空间）
- 蓝图节点网络动态连接
- 材质球在周围漂浮
- 光影扫描效果

**文本（悬停时淡化）：**
- 标题透明度：30%
- 描述：隐藏或透明度 10%
- 技术标签：缩小并淡化

#### HMI前瞻性探索
**视觉效果：**
- 大型数字仪表盘（速度表+转速表）
- 实时变化的速度和RPM数值
- 中控屏幕界面
- HUD导航箭头动画
- 天气效果（雨滴、阳光）

**文本淡化：**
- 只保留项目名称（透明度40%）
- 其他信息淡化或隐藏

#### 智能座舱Niagara
**视觉效果：**
- 全屏粒子系统爆发
- 音波可视化（大型频谱）
- 距离场波纹扩散
- 流光效果贯穿卡片

**文本淡化：**
- 项目名称使用发光效果（透明度50%）
- 其他文本隐藏

#### 比亚迪设计比赛
**视觉效果：**
- 汽车设计线稿到实体渲染过程
- 大型3D奖杯旋转
- 渲染进度条动画
- 设计流程可视化

**文本淡化：**
- 标题保留但缩小（透明度35%）
- 其他文本淡化

#### 局域网FPS游戏
**视觉效果：**
- 游戏准星跟随鼠标
- 射击火花效果
- 游戏UI元素（血条、弹药）
- 网络节点连接可视化

**文本淡化：**
- 保留游戏风格的项目名称
- 其他信息最小化

### 3. 动画实现

#### 悬停进入动画（0.8s）
1. **0-200ms**: 文本开始淡出
2. **200-400ms**: 卡片高度扩展
3. **400-600ms**: 视觉效果元素淡入并放大
4. **600-800ms**: 项目专属动画开始播放

#### 悬停离开动画（等待周期）
1. 检测鼠标离开
2. 等待当前视觉效果完成一个完整动画周期
3. 视觉效果淡出
4. 文本淡入恢复
5. 卡片高度恢复

### 4. 技术细节

#### 文本淡化样式
```css
.project-card:hover .project-title {
  opacity: 0.3;
  transform: scale(0.9);
  transition: all 0.4s ease;
}

.project-card:hover .project-description {
  opacity: 0.1;
  max-height: 0;
  overflow: hidden;
  transition: all 0.4s ease;
}

.project-card:hover .project-tags {
  opacity: 0.25;
  transform: scale(0.8);
  transition: all 0.4s ease;
}
```

#### 视觉效果突出样式
```css
.project-card:hover .visual-effect-container {
  opacity: 1;
  transform: scale(1.2);
  z-index: 10;
}

.visual-effect-container {
  position: absolute;
  inset: 0;
  opacity: 0.3;
  transition: all 0.6s ease;
}
```

### 5. 交互细节

#### 鼠标悬停时
- 文本在0.4s内淡出
- 视觉效果在0.6s内放大并清晰显示
- 背景色渐变到项目主题色

#### 鼠标离开时
- 继续播放当前动画周期
- 周期完成后，视觉效果淡出
- 文本淡入恢复
- 总恢复时间约1-1.5s
