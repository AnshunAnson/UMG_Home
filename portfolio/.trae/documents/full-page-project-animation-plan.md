# 全页面项目动画效果计划

## 需求理解
- 动画效果需要涵盖整个页面，不只是卡片内部
- 卡片太窄，需要更宽的展示空间
- 悬停时界面布局需要动态变化

## 设计方案

### 1. 布局重新设计

#### 默认状态
- 项目卡片全宽显示（100%宽度）
- 每个卡片高度：300px
- 卡片垂直排列，间距：24px
- 卡片内左右分栏：左侧信息（40%）+ 右侧预览（60%）

#### 悬停状态
- 当前卡片扩展到视口高度的80%
- 其他卡片压缩到只剩标题栏（60px高度）
- 被压缩的卡片显示：项目图标 + 标题 + 简短描述
- 当前卡片展开显示完整内容

### 2. 背景系统

#### 页面背景层
- 使用fixed定位覆盖整个视口
- z-index: -1，位于内容下方
- 根据悬停项目动态切换

#### 各项目背景效果
1. **汽车渲染平台**
   - 背景：深青绿色渐变 `#0d0d0d` → `#001a15`
   - 蓝图网格线流动
   - 材质球在角落漂浮
   - 编译着色器进度条

2. **HMI前瞻性探索**
   - 背景：深蓝色渐变 `#0d0d0d` → `#001a2e`
   - 仪表盘元素作为装饰
   - 速度线条流动效果
   - HUD元素闪烁

3. **智能座舱Niagara**
   - 背景：深紫色渐变 `#0d0d0d` → `#1a0a2e`
   - 粒子从中心向外扩散
   - 音波可视化线条
   - 距离场波纹效果

4. **比亚迪设计比赛**
   - 背景：深橙色渐变 `#0d0d0d` → `#2a150a`
   - 设计流程步骤高亮
   - 渲染进度条
   - 奖杯发光效果

5. **局域网FPS游戏**
   - 背景：深粉红色渐变 `#0d0d0d` → `#2a0a15`
   - 准星跟随鼠标
   - 网络节点连线
   - 血条/弹药UI装饰

### 4. 动画时序

#### 悬停进入（0.6s）
1. 0ms: 当前卡片开始扩展
2. 100ms: 其他卡片开始压缩
3. 200ms: 背景开始渐变切换
4. 400ms: 卡片内容重新布局
5. 600ms: 项目专属动画开始播放

#### 悬停离开（等待周期完成）
1. 检测鼠标离开
2. 等待当前动画完成一个完整周期
3. 开始恢复动画（0.6s）
4. 所有卡片恢复到默认状态
5. 背景恢复默认

### 5. 技术实现

#### 状态管理
```typescript
interface ProjectHoverState {
  hoveredId: number | null;
  isAnimating: boolean;
  theme: string;
}
```

#### 组件结构
- `ProjectSection`: 管理整体状态和布局
- `ProjectCard`: 单个项目卡片，支持展开/压缩
- `PageBackground`: 全页面背景层
- `ProjectAnimation`: 项目专属动画组件

#### 动画库
- Framer Motion: 卡片布局动画
- CSS Transitions: 背景渐变
- CSS Animations: 持续循环效果

### 6. 响应式适配

#### 桌面端（>1024px）
- 卡片全宽
- 左右分栏布局
- 完整动画效果

#### 平板端（768px-1024px）
- 卡片全宽
- 上下分栏布局
- 简化动画效果

#### 移动端（<768px）
- 卡片全宽
- 垂直堆叠布局
- 点击展开代替悬停
- 最小化动画效果

### 7. 性能优化

- 使用`will-change`优化动画性能
- 背景层使用GPU加速
- 动画使用`transform`和`opacity`
- 支持`prefers-reduced-motion`
