# 个人作品集页面 redesign 计划

## 问题分析

用户反馈页面"太呆板"、"不像个人主页"。主要问题：

1. **缺少个人特色** - 页面像通用模板，没有体现个人风格
2. **布局死板** - 左右两栏对称布局太常规
3. **内容堆砌** - 简单罗列项目，缺乏故事性
4. **交互单调** - 缺少惊喜和细节

## 改进方向

参考国际顶级设计师个人网站 (如 [soulwire.co.uk](https://soulwire.co.uk/), [p5aholic.me](https://p5aholic.me/))，打造**有灵魂、有记忆点**的个人主页。

## 核心改进

### 1. 首屏 - 强烈的个人标签

**当前**: 普通的标题 + 副标题

**改进**:
- 大字号姓名/昵称，视觉冲击力强
- 一句话个人标签 (不是职位，是身份)
- 动态背景 - 基于UE技能的实时渲染效果
- 鼠标交互产生涟漪/粒子效果
- 滚动提示更优雅

### 2. 关于我 - 有温度的介绍

**新增区域**:
- 不只是技能列表，而是"我是谁"
- 23岁、深圳、3年经验 - 用视觉化方式展示
- 个人照片或头像 (可选)
- 一段有态度的自我介绍，不是简历复制

### 3. 工作经历 - 时间线叙事

**改进**:
- 垂直时间线设计
- 深圳普修汽车资讯有限公司 (2023.04 - 至今)
- 重点突出：汽车渲染项目 (2023.12 - 至今)
- 不是罗列职责，而是展示成果和影响

### 4. 项目展示 - 沉浸式体验

**改进**:
- 汽车渲染项目作为核心展示
- 3D场景与项目内容深度融合
- 材质编写、渲染流水线、性能优化 - 用视觉呈现
- 悬停/点击展开详情
- 不对称布局，打破网格

### 5. 技能 - 可视化展示

**改进**:
- 不是进度条，而是"技能云"或"能力雷达"
- UE4/5、UMG、Material Editor、Niagara 等
- 交互式展示，鼠标悬停显示详情

### 6. 联系 - 有设计感的结尾

**改进**:
- 不只是邮箱和社交链接
- 一句话行动号召
- 磁吸按钮效果
-  footer 有彩蛋

## 设计细节

### 视觉风格
- **主色调**: 深色背景 (#0a0a0f) + 霓虹青绿 (#00d4aa) 作为品牌色
- **字体**: 大标题用粗体无衬线，正文用易读字体
- **动效**: 流畅、有物理感，不拖沓
- **布局**: 留白充足，不对称，有张力

### 交互细节
- 平滑滚动 (Lenis)
- 鼠标跟随效果
- 3D倾斜卡片
- 文字逐字动画
- 视差滚动

### 3D场景优化
- 汽车渲染场景作为核心视觉
- 鼠标移动影响光照/角度
- 后处理效果 (辉光、景深)
- 性能优化，60fps

## 页面结构

```
1. Hero (首屏)
   - 大字号姓名
   - 动态3D背景
   - 一句话介绍
   - 滚动提示

2. About (关于)
   - 个人介绍
   - 关键数据可视化
   - 照片/头像

3. Experience (经历)
   - 时间线
   - 深圳普修汽车资讯
   - 核心成果

4. Projects (项目)
   - 汽车渲染 (重点)
   - UMG重构
   - 其他工作

5. Skills (技能)
   - 可视化展示
   - 交互式

6. Contact (联系)
   - 行动号召
   - 社交链接
   - Footer
```

## 技术实现

### 依赖
- framer-motion (动画)
- gsap + ScrollTrigger (滚动动画)
- lenis (平滑滚动)
- @react-three/fiber (3D)
