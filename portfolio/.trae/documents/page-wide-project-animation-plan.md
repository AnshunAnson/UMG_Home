# 全页面项目动画效果计划

## 需求理解
当鼠标悬停在某个项目卡片上时，整个页面的背景和布局都要发生变化，营造出沉浸式的项目主题氛围。

## 设计方案

### 1. 页面背景变化
- **默认状态**: 深黑色背景 `#0d0d0d` + 细网格线
- **悬停状态**: 根据项目主题改变整个页面背景
  - 汽车项目: 青绿色渐变背景 + 汽车展厅灯光效果
  - HMI项目: 蓝色渐变背景 + 科技感网格
  - Niagara项目: 紫色渐变背景 + 粒子效果
  - 设计项目: 橙色渐变背景 + 设计工作室氛围
  - 游戏项目: 粉红色渐变背景 + 游戏界面元素

### 2. 页面布局变化
- **标题区域**: 项目标题放大并居中显示
- **其他项目卡片**: 淡化或隐藏，突出当前项目
- **项目详情**: 展开显示更多内容
- **视觉元素**: 添加项目相关的装饰元素

### 3. 动画效果
- **过渡动画**: 0.8s 平滑过渡
- **背景动画**: 渐变流动、粒子飘动、光线扫描
- **元素动画**: 缩放、淡入淡出、位移动画
- **鼠标离开**: 等待当前动画周期完成后恢复

### 4. 技术实现
- 使用 React Context 管理全局悬停状态
- Framer Motion 实现页面级动画
- CSS 变量动态切换主题色
- 背景层使用 fixed 定位覆盖整个页面

## 实施步骤

### 阶段1: 创建全局状态管理
1. 创建 ProjectHoverContext
2. 在 layout 中提供 Context
3. 在 Projects 组件中触发状态更新

### 阶段2: 创建页面背景组件
1. 创建 PageBackground 组件
2. 根据悬停项目切换背景
3. 实现渐变、粒子、网格等效果

### 阶段3: 修改页面布局
1. 修改 Projects 组件
2. 悬停时其他卡片淡化
3. 当前卡片放大并居中

### 阶段4: 添加装饰元素
1. 项目专属装饰元素
2. 动态显示/隐藏
3. 与背景协调动画

## 预期效果
- 鼠标悬停项目时，整个页面氛围瞬间改变
- 用户能立即感受到项目的主题和风格
- 专业的视觉冲击力，符合UE5网站水准
