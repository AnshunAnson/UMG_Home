# UE5风格网站重设计计划

## 项目概述
将个人作品集网站改造为具有Unreal Engine 5官方风格的视觉效果，并为每个项目添加专业的悬停动画效果。

## 1. UE5视觉风格设计规范

### 1.1 色彩系统
基于UE5官方网站的配色方案：

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主背景 | `#0d0d0d` | 深黑色，主页面背景 |
| 次背景 | `#1a1a1a` | Slate深灰，卡片背景 |
| 三级背景 | `#242424` | 输入框、按钮背景 |
| 边框色 | `#2a2a2a` | 卡片边框、分割线 |
| 强调色 | `#00d4aa` | UE5标志性青绿色 |
| 强调色hover | `#00b894` | 悬停时的青绿色 |
| 主文字 | `#ffffff` | 白色，标题和重要文字 |
| 次文字 | `#8a8a8a` | 灰色，描述和次要信息 |
| 高亮文字 | `#00d4aa` | 青绿色，链接和强调 |

### 1.2 布局风格
- **网格背景**: UE5编辑器风格的细网格线背景
- **卡片设计**: 带边框的矩形卡片，圆角4-8px
- **间距系统**: 24px基础间距，保持呼吸感
- **字体层次**: 
  - 大标题: 48-64px, font-weight: 700
  - 副标题: 24-32px, font-weight: 600
  - 正文: 16-18px, font-weight: 400

### 1.3 视觉元素
- **渐变效果**: 从`#00d4aa`到`#00a8e8`的线性渐变
- **阴影效果**: 微妙的内阴影和外阴影
- **边框效果**: 1px solid `#2a2a2a`，hover时变为`#00d4aa`
- **玻璃态**: 半透明背景配合backdrop-blur

## 2. 项目卡片悬停效果设计

### 2.1 悬停触发机制
- **触发条件**: 鼠标进入卡片区域
- **动画时长**: 0.6s ease-out
- **离开处理**: 动画完成一个完整周期后恢复(使用animation-fill-mode: forwards)

### 2.2 背景变化效果
1. **整体背景渐变**
   - 从默认深色渐变到项目主题色
   - 使用CSS渐变叠加
   - 透明度从0到0.15

2. **网格背景动画**
   - 网格线颜色从`#2a2a2a`变为项目主题色
   - 网格间距微妙变化
   - 添加流动感

3. **粒子/光效**
   - 根据项目类型添加特定粒子效果
   - 光影扫描效果
   - 边缘发光

### 2.3 布局变化效果
1. **卡片放大**
   - scale: 1 → 1.02
   - z-index提升
   - 阴影加深

2. **内容重排**
   - 标题上移
   - 描述文字展开
   - 技术标签动画进入

3. **详情展示**
   - 悬停时显示项目详情
   - 成就列表滑入
   - 时间线展示

## 6. 技术实现要点

### 6.1 动画技术栈
- **Framer Motion**: 主要动画库，用于React组件动画
- **CSS Animations**: 简单过渡和循环动画
- **CSS Variables**: 动态主题色切换

### 6.2 性能优化
- 使用`will-change`属性
- 动画使用`transform`和`opacity`
- 避免布局抖动
- 使用`requestAnimationFrame`

### 6.3 可访问性
- 支持`prefers-reduced-motion`
- 键盘导航支持
- 屏幕阅读器友好
